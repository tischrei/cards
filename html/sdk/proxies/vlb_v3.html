DLB v3 API¶

For details on how to use database, see /sdk/guides/vlb (NEEDS TO BE DONE)

The VLB Class¶

The database high-level interface is available through the vlb member of a Connection object. The vlb member will only be added if the otcextensions.sdk.register_otc_extensions(conn) method is called.
